Сайт о фильмах / мультфильмах

Домашнее задание: Мини‑сайт о любимых фильмах / сериалах / мультфильмах

Создай свой мини‑сайт. На главной странице будут ссылки на 2–3 внутренние страницы с описанием выбранных произведений.

Шаги

  • Выбери тему: фильмы, сериалы или мультфильмы.
  • Создай папку проекта и файл index.html.
  • Добавь 2–3 страницы в папку pages/.
  • На каждой странице: картинка, описание, список фактов (3 пункта).
  • Добавь стили в style.css: фон, навигация, скругление картинок, тень.

Структура

my_movies/
    index.html
    style.css
    pages/
      film1.html
      film2.html
      film3.html
    img/ (по желанию)

Обязательные теги

<!DOCTYPE html> <html> <head> <meta> <title> <link> <body> <nav> <a> <h1> <p> <img> <ul> <li> <b>

Критерии

  • Работает навигация (2)
  • Главная + 2 страницы (2)
  • Картинка на каждой странице (2)
  • Использованы базовые теги (2)
  • Стили (фон, ссылки, картинки) (2)
  • Аккуратность (+1)

Максимум: 10 (+1 бонус).
