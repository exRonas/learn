HTML‑теги из вашего проекта

Разбор основных элементов, встречающихся в исходных файлах.

Структура страницы

Теги, которые формируют «каркас» страницы.

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8" />
    <title>Заголовок вкладки</title>
    <link rel="stylesheet" href="style.css" />
  </head>
  <body>
    ... содержание страницы ...
  </body>
</html>
  • <html>: корневой элемент страницы.
  • <head>: служебная информация о странице (кодировка, стили).
  • <meta charset>: указывает кодировку текста (UTF‑8).
  • <title>: заголовок во вкладке браузера.
  • <link rel="stylesheet">: подключение CSS‑файла.
  • <body>: видимое содержимое страницы.

Навигация и ссылки

<nav>
  <a href="index.html">Главная</a>
  <a href="about.html">О авторе</a>
  <a href="austalia.html">Австралия</a>
</nav>
  • <nav>: область навигации по сайту.
  • <a>: ссылка на другую страницу (атрибут href).

Заголовки и текст

<h1>Самый большой заголовок</h1>
<h2>Подзаголовок</h2>
<p>Обычный абзац текста.</p>
<p><b>Жирный текст</b> внутри абзаца.</p>
<p>Перенос строки<br>на следующую строку.</p>
  • <h1>…<h6>: заголовки разных уровней.
  • <p>: абзац текста.
  • <b>: полужирное начертание.
  • <br>: перенос строки.

Списки

<ul>
  <li>Пункт 1</li>
  <li>Пункт 2</li>
</ul>
  • <ul>: маркированный список.
  • <li>: пункт списка.

Изображения

<img src="img/picture.jpg" width="300" height="200"
  • <img>: выводит картинку (атрибуты src, width, height).
