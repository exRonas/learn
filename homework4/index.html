Задание Пример работы

Итоговое задание по 4 урокам

Мы с вами прошли 4 важных урока. Вы научились создавать страницы, связывать их ссылками, добавлять картинки и списки, а главное украшать сайт с помощью CSS.

Ваша задача

Создать мини-сайт на свободную тему (любимая игра, фильм, хобби или животное).

Требования (что мы прошли)

  • Структура: Минимум 3 страницы (Главная + 2 страницы с описанием).
  • Навигация: Меню (<nav>) с ссылками на все страницы.
  • Контент: Заголовки (h1), абзацы (p), картинки (img), списки (ul/li).
  • CSS и Классы: Обязательно используйте классы для стилизации. Не стилизуйте просто теги, придумайте классы (например, .red-text, .card, .menu-item).

Какие стили использовать

Попробуйте применить всё, что мы изучили:

  • color и background цвета текста и фона.
  • font-size и font-family шрифты.
  • text-align выравнивание (центр, лево, право).
  • width и height размеры картинок или блоков.
  • margin и padding отступы (внешние и внутренние).

Критерии оценки

  • Сайт состоит из 3+ страниц, ссылки работают 3 балла.
  • Использованы классы в HTML и CSS 3 балла.
  • Страницы выглядят аккуратно (есть отступы, цвета сочетаются) 2 балла.
  • Код написан правильно (теги закрыты, в CSS нет ошибок) 2 балла.
