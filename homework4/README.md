# Домашнее задание: Мини-сайт о фильмах / сериалах / мультфильмах

## Цель
Закрепить базовые HTML‑теги и простые CSS‑стили, научиться делать несколько связанных страниц сайта.

## Что нужно сделать
1. Выберите ТЕМУ: либо фильмы, либо сериалы, либо мультфильмы.
2. Выберите 2–3 произведения (например, 3 любимых мультфильма).
3. Создайте структуру папок:
```
my_movies/
  index.html
  style.css
  pages/
    <первая>.html
    <вторая>.html
    <третья>.html (если нужно)
  img/ (поместите картинки сюда, либо используйте ссылки)
```
4. На главной странице (`index.html`):
   - Заголовок сайта (h1).
   - Краткое описание темы сайта (параграф p).
   - Список ссылок (a) на ваши страницы.
5. На каждой внутренней странице:
   - Заголовок (h1).
   - Картинка (`img`) постера или кадра.
   - Абзац описания (`p`).
   - Список из 3 фактов (`ul` + `li`).
6. Добавьте простые стили в `style.css`:
   - Фон сайта (`background`).
   - Стили для навигации (`nav a`).
   - Скругление углов картинок (`border-radius`).
   - Тень для карточек (`box-shadow`).

## Обязательные теги
`<!DOCTYPE html>`, `html`, `head`, `meta`, `title`, `link`, `body`, `nav`, `a`, `h1`, `h2` (если нужно), `p`, `img`, `ul`, `li`, `b` (можно для выделения).

## Дополнительные идеи (по желанию)
- Выделить важные слова стилем (класс `.highlight`).
- Сделать общий блок фактов с лёгким фоном.
- Добавить внизу футер.

## Критерии оценки
| Критерий | Баллы |
|----------|-------|
| Работает навигация между страницами | 2 |
| Есть главная и минимум 2 контентные страницы | 2 |
| На каждой странице есть изображение | 2 |
| Использованы перечисленные базовые теги | 2 |
| Есть собственные стили (фон + оформление ссылок/картинок) | 2 |
| Дополнительно (аккуратность, читаемость) | +1 |

Максимум: 10 (+1 бонус)

## Как отправить
Сожмите папку проекта в архив (zip) и отправьте преподавателю. Или загрузите на бесплатный хостинг и пришлите ссылку.
